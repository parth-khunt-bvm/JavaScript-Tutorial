JavaScript Tutorial - data types

A data type is an attribute associated with a piece of data that tells a computer system how to interpret its value.

  • Primitives Data-type
    • Null
      Ex: let nullType = null;
    • Number
      Ex: let numberType = 10;
    • Boolen
      Ex: let boolenType = null;
    • Bigint
      Ex: let bigintType = BigInt("576");
    • Symbol
      Ex: let symbolType = Symbol("This is symbol data type");
    • String
      Ex: let stringType = "Hello, Good Evening";
    • Undefined
      Ex: let undefinedType = undefined ;
  • Object Data-type
    • Object
      let student = { name : 'Parth Khunt', age : 27, city : 'Surat', dob : '10-April-1996', };
